## Idea 1: "Nordic Industrial Precision"

**Design Movement:** Scandinavian Industrial Minimalism — inspired by Atlas Copco's Swedish heritage. Clean, functional, authoritative.

**Core Principles:**
1. Functional clarity — every element earns its space
2. Restrained color palette with strategic accent pops
3. Dense information display without visual clutter
4. Typography-driven hierarchy

**Color Philosophy:** Deep navy (#0d3b66) as the anchor of authority and trust. Warm gold (#d4a843) as the accent — referencing Atlas Copco's brand identity. Teal (#0d7c8c) for interactive elements and data highlights. Neutral warm grays for backgrounds. The palette communicates "serious industrial intelligence" without feeling cold.

**Layout Paradigm:** Persistent top navigation with tabbed content areas. Full-width data tables and card grids. Left-aligned content with generous horizontal padding. No sidebar — the data IS the interface.

**Signature Elements:**
1. Colored left-border cards that indicate priority (red=hot, amber=warm, grey=cold)
2. Evidence badges with confidence indicators (verified/predicted/unknown)
3. Expandable project cards with structured detail grids

**Interaction Philosophy:** Click-to-expand for progressive disclosure. Inline filtering without page reloads. Sortable tables with visual sort indicators. Smooth accordion animations.

**Animation:** Subtle card hover lifts (translateY -2px). Smooth accordion expand/collapse (300ms ease). Tab transitions with content fade. No gratuitous motion — every animation serves information access.

**Typography System:** DM Sans for headings (geometric, modern, industrial feel). Inter for body text and data tables (maximum legibility at small sizes). Monospace for numerical data in KPI cards.

## Idea 2: "Command Center Dark"

**Design Movement:** Dark-mode data operations center — inspired by military/aerospace command dashboards.

**Core Principles:**
1. Dark background reduces eye strain for daily monitoring
2. Neon accent colors for urgency and priority signaling
3. Grid-heavy layout with modular data panels
4. Real-time operations feel

**Color Philosophy:** Near-black (#0a0f1a) background. Electric blue (#00d4ff) for primary actions. Amber (#ffb800) for warnings. Red (#ff3b3b) for hot/urgent. Green (#00ff88) for confirmed/positive. The palette creates urgency and operational awareness.

**Layout Paradigm:** Full-bleed dark canvas. Floating card panels with subtle borders. Dashboard grid with resizable panels. Persistent sidebar navigation.

**Signature Elements:**
1. Glowing border effects on priority cards
2. Animated pulse indicators on "hot" projects
3. Terminal-style data readouts for KPIs

**Interaction Philosophy:** Hover reveals additional data layers. Click-through drill-down navigation. Real-time filter updates with count badges.

**Animation:** Subtle glow pulses on hot items. Smooth panel transitions. Data counter animations on load. Typing effect for executive summary.

**Typography System:** JetBrains Mono for data/numbers. Space Grotesk for headings. Inter for body text.

## Idea 3: "Executive Briefing"

**Design Movement:** Premium editorial design — inspired by Bloomberg Terminal meets McKinsey report aesthetics.

**Core Principles:**
1. Information density balanced with visual breathing room
2. Professional credibility through restrained elegance
3. Print-quality typography and data presentation
4. Clear visual hierarchy guiding the eye from summary to detail

**Color Philosophy:** White/off-white canvas for maximum readability. Deep charcoal (#1a1a2e) for text authority. Atlas Copco teal (#0d7c8c) as the primary brand accent. Warm gold (#d4a843) for premium highlights. Status colors (red/amber/grey) used sparingly and only for classification. The palette says "boardroom-ready intelligence."

**Layout Paradigm:** Magazine-style asymmetric layout. Wide content column with narrow metadata sidebar. Section breaks with horizontal rules and typographic headers. Tables as first-class citizens, not afterthoughts.

**Signature Elements:**
1. Oversized KPI numbers with micro-label typography
2. Horizontal colored rules under section headers (matching priority)
3. Citation-style source references with dates

**Interaction Philosophy:** Progressive disclosure through expandable sections. Keyboard-navigable tables. Quick-filter chips that feel like editorial tags.

**Animation:** Elegant fade-in on scroll. Smooth table sort transitions. Subtle hover state changes. Card expansion with spring physics.

**Typography System:** Playfair Display for the main title/brand. DM Sans for section headers and navigation. Inter for body, tables, and data. Clear size hierarchy: 28px → 18px → 14px → 12px → 10px.
