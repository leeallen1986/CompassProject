# Platform navigation shell

## Purpose

This change makes the commercial operating sequence visible across the primary sales surfaces without changing project ranking, Full Potential data or pursuit logic.

The primary navigation is now:

1. **This Week** — the rep's operating home and weekly action board.
2. **Explore Projects** — broad project, contractor, award and tender intelligence.
3. **Full Potential** — canonical account universe and commercial-potential workflow.
4. **Account Intelligence** — account-centred project, contractor and stakeholder context.
5. **Pursuits** — early attributed pursuits before formal C4C qualification.

My Working Style, Settings and Admin are secondary user-menu items rather than primary commercial destinations.

## Pump Targets boundary

The previous **WA Targets** label referred to a Pump/Flow account-prior library. It is renamed **Pump Targets** and is visible only when:

- the user is an admin; or
- an assigned business line contains Pump, Dewatering or Flow.

Portable Air-only users are directed to Full Potential instead. The route and source data remain available to the Pump and Dewatering team; this release does not delete or migrate them.

## Route shell

The shared navigation shell wraps:

- This Week;
- Explore Projects;
- Account Intelligence;
- Pursuits;
- Pump Targets;
- Full Potential and its pilot, modelling, segment and data-quality subroutes.

Legacy page headers are hidden where they only duplicated global navigation. Page-specific controls remain visible, including Full Potential import/review controls and the Pursuits mine/team selector.

## Commercial boundary

This release changes navigation, access presentation and page labels only. It does not alter:

- project scoring or visibility;
- contractor inference;
- Full Potential matching;
- account ownership or routing;
- model, evidence or action records;
- pipeline claims or stages;
- schemas or migrations;
- C4C data.

The next platform-audit release should create one reusable project/contractor-to-Full-Potential account resolution service before the Explore Projects redesign.
